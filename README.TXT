Boondoggle

Multiplayer mini-games to be played between students via our schools intranet 

* Rufen Sie in der Klasse VierGewinntFeld die main Methode auf
* daraufhin öffnet sich ein Fenster:
    * Ein Hintergrundbild und ein 4 Gewinnt Feld sollte erscheinen
* Nun können Sie sowohl für Spieler 1(rot) als auch
  für Spieler 2(gelb) Chips platzieren indem:
    * Sie mit der linken Maustaste auf ihrem gewünschtem Feld einen Chips
      platzieren
    * dieser wird automatisch so weit unten wie möglich platziert
* Wenn Sie das Spiel beenden wollen drücken Sie auf das "x"
